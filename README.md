# Automation SEO Audit Tech

Application Streamlit publique pour analyser des exports Screaming Frog et produire un audit technique SEO deterministe.

## Deploiement Streamlit Community Cloud

- Repository : `YN-CodingClub/automation-seo-audit-tech`
- Branch : `main`
- Main file path : `streamlit_app.py`
- App URL : `automation-seo-audit-tech`

URL cible :

```text
https://automation-seo-audit-tech.streamlit.app/
```

## Donnees

L'application attend un export CSV Screaming Frog fourni par l'utilisateur. Aucun output client ni log local n'est embarque dans ce repo public.
